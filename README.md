# 照明卓シミュレーター

舞台照明のフェーダーをシミュレーションするiPad向けのWebアプリケーションです。

## 特徴

- **iPad最適化**: 縦画面での使用を想定したレスポンシブデザイン
- **リアルタイムプレビュー**: フェーダー操作に応じたライトの視覚化
- **系統別管理**: A系統・B系統に分けられた36チャンネルのフェーダー
- **光の三原色対応**: RGB、アンバー、青緑、黄、青紫などの色彩表現
- **フラッシュ機能**: 各フェーダーに独立したフラッシュボタン

## 構成

### プレビューエリア
- 舞台の簡略化された3Dビュー
- リアルタイムでのライト強度・色の表示
- 光の三原色による加法混色シミュレーション

## 技術仕様

- **Framework**: Next.js 15.5.3
- **UI**: Tailwind CSS
- **Language**: TypeScript
- **対応デバイス**: iPad (縦画面推奨)

## セットアップ

```bash
npm install
npm run dev
```

http://localhost:3000 でアプリケーションにアクセスできます。

## 使用方法

1. フェーダーを上下にドラッグして光の強度を調整
2. FLASHボタンで瞬間的な点灯
3. プレビューエリアで視覚的な確認
4. A系統・B系統を独立して操作可能

## iPad での使用

iPadのSafariでアクセスし、ホーム画面に追加することでネイティブアプリのような体験が可能です。
